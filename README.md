# Hello World Program

As a software engineer, I'm building foundational skills in version control with Git and GitHub while establishing a professional development workflow. This project serves as my starting point for learning how to properly document, version, and share software projects.

## Description
A simple command-line program that prints "Hello World" to the console. This program demonstrates the basic structure of a runnable application and serves as a verification tool to ensure my development environment (Python, Git, VS Code) is correctly configured.

## Purpose
I created this software to establish a repeatable workflow for all future programming projects. By starting with a simple "Hello World" program, I can focus on mastering essential software engineering practices including Git commands, README documentation standards, code walkthroughs, and video demonstrations without the distraction of complex program logic.
